# Simple Password Protection

A Christmas-themed password strength checker and security education tool.

## Features

- Password strength analysis
- Real-time feedback on password requirements
- Educational content about password security
- Festive Christmas design with animated lights

## Live Demo

Visit the live site at: [https://neqkuny.github.io/secure-password-hub/](https://neqkuny.github.io/secure-password-hub/)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build creates a single, self-contained HTML file with all assets inlined.
